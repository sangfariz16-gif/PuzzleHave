# 🧩 PuzzleHave — Sliding Puzzle Game

Game sliding puzzle berbasis web yang bisa dimainkan langsung di browser!

🎮 **Live Demo:** [sangfariz16-gif.github.io/PuzzleHave](https://sangfariz16-gif.github.io/PuzzleHave)

## ✨ Fitur

- 🖼️ **8 Gambar Berbeda** — Sunset, Hutan, Lautan, Gunung, Kota, Sakura, Luar Angkasa, Abstrak
- 🎯 **3 Tingkat Kesulitan** — 3×3 (Mudah), 4×4 (Normal), 5×5 (Sulit)
- 🏆 **Leaderboard Lokal** — Simpan skor dan bersaing dengan dirimu sendiri
- 📖 **Tutorial Interaktif** — 5 slide panduan bermain
- 🎨 **6 Tema Visual** — Default, Ocean, Forest, Sunset, Dark, Candy
- 💡 **Sistem Petunjuk** — Minta bantuan saat bingung
- 👁️ **Lihat Asli** — Tampilkan gambar lengkap sementara
- ⌨️ **Keyboard Support** — Geser dengan tombol panah
- 📱 **Responsif** — Bekerja di mobile dan desktop

## 🎮 Cara Bermain

1. Pilih gambar dan tingkat kesulitan
2. Klik keping yang bersebelahan dengan ruang kosong untuk menggesernya
3. Susun semua keping hingga membentuk gambar utuh
4. Selesaikan secepat mungkin dengan langkah sesedikit mungkin!

## 🛠️ Teknologi

- **HTML5** — Struktur halaman
- **CSS3** — Animasi & tema
- **JavaScript Vanilla** — Logika game (tanpa framework)
- **Canvas API** — Render gambar puzzle
- **GitHub Pages** — Hosting gratis

## 📦 Cara Menjalankan Secara Lokal

```bash
# Clone repository
git clone https://github.com/sangfariz16-gif/PuzzleHave.git

# Masuk ke direktori
cd PuzzleHave

# Buka dengan browser (bisa langsung buka index.html)
# atau gunakan server lokal:
npx serve .
```
